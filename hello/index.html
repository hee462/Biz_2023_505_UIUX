처음으로 만드는 web Page

처음으로 만드는 HTML 문서로 작성된 Web Page, Web Page는 HTML 형식으로 작성한다 HTML 문서에서 p tag로 시작하면 하나의 문단이 시작된다. 하나의 문단은 close/p 이며 문단이 끝났음을 나타낸다 문단 내에서는 아무리 Enter를 입력하여도 Browser에 나타날때는 모두 무시된다 단 Enter를 입력하면 한칸의 빈칸이 나타낸다. 문단 중가에서 여러개의 빈칸이 (Space)를 입력하여도 한개의 빈칸만 남기고 모두 무시된다.

HTML 문서는 tag로 감싸진 Text로 구성되어있다 tag로 감싸진 문서를 작성하고 Web Browser를 통하여 확인한다
