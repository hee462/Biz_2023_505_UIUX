# HTML에서 List 표현하기

- 순서를 명시하지 않고 List를 나열하는 방법
- 순서를 보여주면서 List를 나열하는 방법
- List를 표현하기 위한 box tag를 만들고, box tag에 item tag를 배열하여 List를 표현한다
- List box tag 와 List item tag는 모두 block속성을 갖는다 -> 세로방향 작성만 가능

## 순서가 없는 List: ul

```html
<ul>
  <li>item1</li>
  <li>item2</li>
  <li>item3</li>
</ul>
```

## 순서가 있는 List:ol

```html
<ol>
  <li>item1</li>
  <li>item2</li>
  <li>item3</li>
</ol>
```
